# ARIMA 기본 용어

## 결론부터

**ARIMA는 추세를 차분으로 줄인 뒤, 과거 값과 과거 오차를 이용해 현재 또는 미래 값을 설명하는 시계열 모델입니다.**

---

## 정상성

정상성은 시계열의 평균, 분산, 자기상관 구조가 시간이 지나도 크게 변하지 않는 상태입니다.

비정상적인 예:

| 시간 | 온도 |
|---|---:|
| 09:00 | 70.0 |
| 09:01 | 70.5 |
| 09:02 | 71.0 |
| 09:03 | 71.5 |
| 09:04 | 72.0 |
| 09:05 | 72.5 |

계속 올라가고 있으므로 평균이 시간에 따라 변합니다.

비교적 정상적인 예:

| 시간 | 온도 |
|---|---:|
| 09:00 | 70.1 |
| 09:01 | 69.9 |
| 09:02 | 70.2 |
| 09:03 | 70.0 |
| 09:04 | 69.8 |
| 09:05 | 70.1 |

70도 근처에서 흔들릴 뿐입니다.

---

## 차분

차분은 현재 값에서 바로 이전 값을 빼는 것입니다.

| 시간 | 온도 | 1차 차분 |
|---|---:|---:|
| 09:00 | 70.0 | - |
| 09:01 | 70.5 | 0.5 |
| 09:02 | 71.0 | 0.5 |
| 09:03 | 71.4 | 0.4 |
| 09:04 | 71.8 | 0.4 |
| 09:05 | 72.1 | 0.3 |

원래 온도는 계속 증가하지만, 차분값은 0.3~0.5 근처에서 움직입니다. 즉, “온도 수준”이 아니라 “온도가 얼마나 빨리 변하는지”를 보는 것입니다.

---

## AR

AR은 AutoRegressive, 즉 자기회귀입니다.

**현재 값은 과거 값의 영향을 받는다**는 생각입니다.

가장 단순한 AR(1)은 이렇게 씁니다.

\[
y_t = c + \phi y_{t-1} + \epsilon_t
\]

| 기호 | 의미 |
|---|---|
| \(y_t\) | 현재 시점의 값 |
| \(y_{t-1}\) | 바로 이전 시점의 값 |
| \(c\) | 기본 수준 |
| \(\phi\) | 이전 값이 현재에 얼마나 영향을 주는지 |
| \(\epsilon_t\) | 설명하지 못한 오차 |

설비 온도, 압력, 전류 RMS처럼 천천히 움직이는 센서에서는 AR 성분이 강할 수 있습니다.

---

## MA

MA는 Moving Average입니다.

ARIMA에서 MA는 단순 rolling average가 아니라 **과거의 예측 오차가 현재 값에 영향을 준다**는 뜻입니다.

가장 단순한 MA(1)은 이렇게 씁니다.

\[
y_t = c + \epsilon_t + \theta \epsilon_{t-1}
\]

| 기호 | 의미 |
|---|---|
| \(\epsilon_t\) | 현재 오차 |
| \(\epsilon_{t-1}\) | 바로 이전 오차 |
| \(\theta\) | 이전 오차의 영향력 |

---

## ARIMA

ARIMA는 세 가지를 합친 이름입니다.

\[
ARIMA(p, d, q)
\]

| 요소 | 의미 | 쉬운 설명 |
|---|---|---|
| \(p\) | AR 차수 | 과거 값을 몇 시점까지 볼 것인가 |
| \(d\) | differencing 차수 | 몇 번 차분할 것인가 |
| \(q\) | MA 차수 | 과거 오차를 몇 시점까지 볼 것인가 |

예를 들어 ARIMA(1,1,1)은 이런 뜻입니다.

1. 원본 데이터를 한 번 차분한다.
2. 차분된 값에 대해 과거 1시점 값을 본다.
3. 과거 1시점의 오차도 반영한다.

---

## ACF와 PACF

ACF는 자기상관함수입니다.

**지금 값은 1분 전, 2분 전, 3분 전 값과 얼마나 닮았나?**

PACF는 부분 자기상관함수입니다.

**1분 전의 영향을 제거하고도, 2분 전 값이 현재와 직접 관련이 있나?**

| 도구 | 보는 것 | 도움 되는 부분 |
|---|---|---|
| ACF | 과거 값들과의 전체 상관 | MA 후보 차수 \(q\) |
| PACF | 중간 lag 영향을 제거한 직접 상관 | AR 후보 차수 \(p\) |

입문 단계에서는 ACF/PACF를 완벽히 외우기보다, “과거 몇 시점까지 의미 있는 기억이 남아 있는지 보는 도구”라고 이해하면 충분합니다.
